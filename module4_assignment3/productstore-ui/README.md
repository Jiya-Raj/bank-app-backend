## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Update backend URL if required in `src/environments/environment.ts`.
3. Start app:
   ```bash
   npm start
   ```

The UI fetches data from:

- `GET /products`

Default backend base URL:

- `http://localhost:8080`
